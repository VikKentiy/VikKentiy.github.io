### DialogBox

Реализовано классом.


### Описание Демки
При клике на соответствующие кнопки - появляется и исчезает.
При кликах на checkbox(ы) появляется input с подобающей фразой.


### Использование

```javascript

var
    dialog = new Dialog('#cssSelector');

    dialog.show(function onShow(){
        dialog.hide();
    })

```
